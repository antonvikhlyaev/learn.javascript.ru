Какой скрипт выполнится первым?

В примере ниже подключены два скрипта small.js и big.js.

Если предположить, что small.js загружается гораздо быстрее, чем big.js – какой выполнится первым?

    script src="big.js"
    script src="small.js"

Ответ: сначала big.js, затем small.js по стандартной последовательности загрузки скриптов.

А вот так?

    script async src="big.js"
    script async src="small.js"

Ответ: сначала small.js, затем big.js потому, что скрипты загружаются асинхронно и первым будет тот, кто загрузился быстрее.

А так?

    script defer src="big.js"
    script defer src="small.js"

Ответ: сначала big.js, потом small.js. С атрибутом defer порядок загрузки скриптов сохраняется, независимо от скорости загрузки.
